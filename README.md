# Liquid Glass +
Enhanced Liquid Glass theme for "Better Lyrics" Chrome extension

# How To Install
In a text editor, open the `.CSS` file, copy everything, and paste it into Better Lyrics' "Custom Style" (Lyrics for YouTube Music).
